10. ```python
    # 字符串的切片操作会返回一个新字符串
    
    a = "ABCDE"
    
    print(a[1:4])
    # 输出 BCD
    print(a[1:])
    # 输出 BCDE
    print(a[:4])
    # 输出ABCD
    print(a[:])
    # 输出ABCDE
    
    # a[begin:end] 会返回包含a[begin], a[begin + 1], ..., a[end - 1]的字符串
    # 由于切片操作返回的仍是字符串，故不支持修改操作
    
    # a[1:4] = "XY"
    # 会报错，字符串不能单独对某一部分修改
    ```
