##### 递归与分治

1. 画出示意图

![](D:\Study\myAlgorithm\算法笔记\搜索与图论\DFS与BFS\AcWing_3695_扩充序列\扩充序列示意图.bmp)

2. 我们要得出 u 次扩充后，序列第 k 个元素的值

3. 所以我们考察第 u 次扩充后，序列中元素的分布情况

   - 左侧

     （1）第 k 个元素的值与 u - 1 次扩充第 k 个元素值相同

     （2）将问题转换为求 u - 1 次扩充后，第 k 个元素值

   - 中间

     （1）第 k 个元素值即为 u + 1

   - 右侧

     （1）第 k 个元素的值与 u - 1 次扩充第 2 ** (u + 1) 个元素值对称

     （2）将问题转换为求 u - 1 次扩充后，第 2 ** (u + 1) - k 个元素值

4. 写出递归代码

```python
def f(u, k):
    if k > 2 ** u:
        return f(u - 1, 2 ** (u + 1) - k)
        # 位于右侧：对称到上一轮的位置

    elif k == 2 ** u:
        return u + 1
        # 位于中间：直接返回结果

    else:
        return f(u - 1, k)
        # 位于左侧：上一轮的相同位置
```

### 详细代码

```python
def f(u, k):
    # u：扩充第 u 次
    # k：第 k 个数字

    if k > 2 ** u:
        return f(u - 1, 2 ** (u + 1) - k)
        # 位于右侧：对称到上一轮的位置

    elif k == 2 ** u:
        return u + 1
        # 位于中间：直接返回结果

    else:
        return f(u - 1, k)
        # 位于左侧：上一轮的相同位置


n, k = map(int, input().split())
# 读取需要扩充 n - 1 次与第 k 个数字

# 递归调用
res = f(n - 1, k)
print(res)
```
